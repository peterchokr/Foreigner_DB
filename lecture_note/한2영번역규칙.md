### Rule 3: 특징과 설명 (Characteristics & Descriptions)

#### ✅ 올바른 방식

```markdown
**1. Automatic Execution (자동 실행)**

- No need for user to explicitly call (사용자가 명시적으로 호출할 필요가 없음)
- Automatically activates when INSERT/UPDATE/DELETE occurs (INSERT/UPDATE/DELETE가 발생하면 자동으로 작동)

**2. Ensure Data Integrity (데이터 무결성 보장)**

- Prevent incorrect data input (잘못된 데이터 입력 방지)
- Automatically validate data rules (데이터 규칙 자동 검증)
```

### Rule 4: SQL 코드 주석

#### ✅ 올바른 방식

```sql
-- Create table (테이블 생성)
CREATE TABLE employees (
    employee_id INT PRIMARY KEY AUTO_INCREMENT,
    -- Employee name (직원 이름)
    name VARCHAR(50),
    -- Employee salary (직원 급여)
    salary DECIMAL(10, 2)
);

-- Insert sample data (샘플 데이터 삽입)
INSERT INTO employees VALUES
(1, 'Alex Johnson', 5000000),
(2, 'Sarah Williams', 4000000);
```

**패턴:**

- 주요 작업 설명: 정열식 주석 `-- English (한글)`
- 라인별 설명: 같은 형식 사용
- 한글은 항상 괄호 안에

#### ✅ 복잡한 쿼리

```sql
-- Trigger to track salary changes (급여 변경을 추적하는 트리거)
DELIMITER //
CREATE TRIGGER salary_update_log
AFTER UPDATE ON employees
FOR EACH ROW
BEGIN
  -- Check if salary actually changed (급여가 실제로 변경되었는지 확인)
  IF NEW.salary != OLD.salary THEN
    INSERT INTO salary_history 
    (emp_id, emp_name, old_salary, new_salary, changed_at)
    VALUES (NEW.employee_id, NEW.name, OLD.salary, NEW.salary, NOW());
  END IF;
END //
DELIMITER ;
```

### Rule 6: 표 (Tables)

#### ✅ 올바른 방식

```markdown
| Function | Description |
|:--------:|:-----------|
| `sqlite3.connect()` | Connect database (데이터베이스 연결) |
| `cursor.execute()` | Execute one SQL (SQL 한 개 실행) |
| `cursor.fetchall()` | Return all rows (모든 행 반환) |
| `conn.commit()` | Save changes (변경사항 저장) |
```

**패턴:**

- 헤더: 영어만
- 셀 내용: `함수명` (영문 설명 한글 설명)
- 괄호로 영문과 한글 매핑

#### ✅ 복잡한 표

```markdown
| Operation | NEW Available | OLD Available |
|-----------|---------------|---------------|
| INSERT | ✓ Yes | ✗ No |
| UPDATE | ✓ Yes | ✓ Yes |
| DELETE | ✗ No | ✓ Yes |
```

### Rule 7: 코드 블록 (Code Blocks)

#### ✅ 올바른 방식

```python
import sqlite3

# Create database connection (데이터베이스 연결 생성)
conn = sqlite3.connect('todo.db')
cursor = conn.cursor()

# Create table (테이블 생성)
sql = '''
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL
)
'''

cursor.execute(sql)
conn.commit()  # Save changes (변경사항 저장)
conn.close()
```

**패턴:**

- 각 블록의 첫 주석: `-- English (한글)`
- 인라인 주석: 코드 끝에 `# English (한글)`
- 모든 주석은 이중언어
